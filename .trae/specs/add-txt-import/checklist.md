# TXT/MD 导入功能 — 验证检查清单

## 单章导入
- [x] 右键章节节点 → "从文件导入"，选 .txt/.md 后内容替换
- [x] 编辑器即时刷新，字数统计更新
- [x] 文件不存在/读取失败时提示错误

## 分卷批量导入
- [x] 右键分卷 → "从文件夹导入章节"
- [x] 按文件名自然排序创建章节
- [x] 文件名（无扩展名）作为章节标题
- [x] 空目录/无 .txt/.md 文件时提示
- [x] 导入完成后显示结果报告

## 全本自动分割导入
- [x] 按 `## 第X章` 分割生效
- [x] 按 `=== 第X章 ===` 分割生效
- [x] 按 `\n\n\n` 分割生效
- [x] 分割后每个段落标题正确提取

## 导入预览对话框
- [x] 显示分割段落列表（标题 + 字数）
- [x] 右侧显示段落内容预览
- [x] 可选择目标分卷（已有 + 新建）
- [x] 确认后批量创建章节

## 菜单集成
- [x] 文件 → 导入 → 从 TXT/MD 导入
- [x] 右键章节 → 从文件导入
- [x] 右键分卷 → 从文件夹导入章节
