# Testing Guide

> **项目版本**: v0.20.0  
> **最后更新**: 2026-07-02  
> **测试状态**: 暂无自动化测试，以下为手动测试清单

---

## 1. 概述

本项目是一个 CLI 工具，目前**没有自动化测试**。所有测试需要手动执行。

**测试环境要求**:
- Node.js >= 18.0.0
- npm >= 8.x
- 至少一个 AI 助手（Claude Code、Cursor、Gemini CLI 等）

**测试准备**:
```bash
# 进入项目目录
cd /path/to/novel-writer-2

# 安装依赖
npm install --ignore-scripts

# 构建项目
npm run build
npm run build:commands
```
