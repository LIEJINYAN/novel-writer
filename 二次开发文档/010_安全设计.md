# 安全设计

## 1. 安全设计原则

PySide6 版作为桌面应用，安全设计与 Web 应用有本质区别。桌面应用的攻击面较小，但需要关注以下核心安全原则：

1. **本地优先**：敏感数据不离开用户设备
2. **加密存储**：API Key 等凭证加密存储
3. **最小权限**：只读取必要的数据
4. **透明可控**：用户完全掌握自己的数据
5. **无远程调用**：除非用户主动配置 AI API

## 2. 安全风险评估

### 2.1 风险矩阵

| 风险类型 | 严重程度 | 发生概率 | 风险等级 | 说明 |
|---------|---------|---------|---------|------|
| API Key 泄露 | 🔴 高 | 🟡 中 | 🔴 高 | 通过配置文件或日志泄露 |
| 提示词注入 | 🟡 中 | 🟢 低 | 🟡 中 | 恶意提示词构造 |
| 数据库文件泄露 | 🟡 中 | 🟢 低 | 🟡 中 | 物理访问导致 |
| 插件恶意代码 | 🟡 中 | 🟢 低 | 🟡 中 | 第三方插件执行 |
| 网络嗅探（API 调用） | 🟢 低 | 🟢 低 | 🟢 低 | 默认 HTTPS |
| 内存敏感数据残留 | 🟢 低 | 🟡 中 | 🟢 低 | 解密后 Prompt 在内存中 |

### 2.2 与原版风险对比

| 风险维度 | 原版（CLI + AI 助手） | PySide6 版（桌面应用） |
|---------|---------------------|---------------------|
| API Key 存储 | 环境变量 | 加密存储到 SQLite |
| 网络通信 | HTTPS（仅外部插件） | HTTPS（所有 API） |
| 数据隔离 | 文件系统 | 文件系统 + 数据库 |
| 输入验证 | 基本无 | 严格（服务端控制） |
| 插件权限 | 完全信任 | 可控制（启用/禁用） |
| 内存安全 | JavaScript（字符串不可变） | Python（可显式清理） |
| 日志敏感信息 | 可能泄露 | 自动脱敏 |

## 3. API Key 安全存储

### 3.1 加密存储方案

```python
# core/security/vault.py
import os
import base64
import json
from cryptography.fernet import Fernet
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC

class CredentialVault:
    """凭证保险库 - 加密存储 API Key 等敏感信息"""
    
    def __init__(self, vault_path: str):
        self.vault_path = vault_path
        self._key = self._load_or_create_key()
        self._fernet = Fernet(self._key)
    
    def _get_machine_key(self) -> bytes:
        """生成基于设备的加密密钥"""
        import getpass
        import platform
        
        # 使用机器特征 + 随机盐值生成密钥
        machine_info = (
            platform.node() +
            getpass.getuser() +
            self._get_salt()
        )
        
        kdf = PBKDF2HMAC(
            algorithm=hashes.SHA256(),
            length=32,
            salt=self._get_salt_bytes(),
            iterations=600000,
        )
        return base64.urlsafe_b64encode(kdf.derive(machine_info.encode()))
    
    def _get_salt(self) -> str:
        """获取随机盐值（首次运行时生成并存储）"""
        salt_file = os.path.join(os.path.dirname(self.vault_path), '.salt')
        if os.path.exists(salt_file):
            with open(salt_file, 'r') as f:
                return f.read().strip()
        
        # 首次运行，生成随机盐值
        import secrets
        salt = secrets.token_hex(32)
        with open(salt_file, 'w') as f:
            f.write(salt)
        return salt
    
    def store_api_key(self, provider: str, api_key: str):
        """加密存储 API Key"""
        encrypted = self._fernet.encrypt(api_key.encode())
        
        vault = self._load_vault()
        vault[provider] = {
            "key": encrypted.decode(),
            "created_at": datetime.now().isoformat(),
        }
        self._save_vault(vault)
    
    def get_api_key(self, provider: str) -> str:
        """解密获取 API Key"""
        vault = self._load_vault()
        if provider not in vault:
            raise KeyError(f"未找到 {provider} 的 API Key")
        
        encrypted = vault[provider]["key"].encode()
        return self._fernet.decrypt(encrypted).decode()
    
    def has_api_key(self, provider: str) -> bool:
        """检查是否存在 API Key"""
        vault = self._load_vault()
        return provider in vault
    
    def delete_api_key(self, provider: str):
        """删除 API Key"""
        vault = self._load_vault()
        vault.pop(provider, None)
        self._save_vault(vault)
```

### 3.2 密钥管理

```
密钥派生流程：

设备信息 (主机名 + 用户名) + 随机盐值
                  ↓
          PBKDF2 (600,000 次迭代)
                  ↓
        32 字节加密密钥 (AES-256)
                  ↓
         Fernet (AES-256-CBC + HMAC)
```

**安全特性**：
- 每个安装实例使用不同的随机盐值
- API Key 在数据库中加密存储（非明文）
- 密钥通过 PBKDF2 派生，迭代 60 万次
- API Key 仅在使用时解密到内存，用后清除

## 4. 提示词注入防护

### 4.1 输入验证

```python
# core/ai/input_validator.py
import re
from typing import Optional

class PromptInputValidator:
    """提示词输入验证器"""
    
    # 禁止的系统指令关键词
    FORBIDDEN_PATTERNS = [
        r"ignore\s+(all\s+)?(previous|above|prior)",
        r"disregard\s+(all\s+)?(previous|above)",
        r"forget\s+(all\s+)?(previous|prior)",
        r"你(是|现在(是|扮演))",
        r"you\s+(are|act\s+as)",
        r"system\s*:?\s*you\s+are",
        r"role\s*:?\s*assistant",
    ]
    
    # 最大输入长度
    MAX_INPUT_LENGTH = 50000  # 字符
    MAX_PARAM_LENGTH = 1000   # 单个参数
    
    @classmethod
    def validate_prompt_input(cls, user_input: str) -> tuple[bool, Optional[str]]:
        """验证用户输入"""
        # 1. 长度检查
        if len(user_input) > cls.MAX_INPUT_LENGTH:
            return False, f"输入过长（最长 {cls.MAX_INPUT_LENGTH} 字符）"
        
        # 2. 禁止模式检查
        for pattern in cls.FORBIDDEN_PATTERNS:
            if re.search(pattern, user_input, re.IGNORECASE):
                return False, "输入包含不安全的指令模式"
        
        # 3. 命令注入检查
        if re.search(r"[;&|`$(){}]", user_input):
            return False, "输入包含不允许的特殊字符"
        
        return True, None
    
    @classmethod
    def sanitize_prompt_input(cls, user_input: str) -> str:
        """清理输入"""
        # 移除潜在的注入内容
        sanitized = user_input
        
        # 移除多余空白
        sanitized = re.sub(r'\s+', ' ', sanitized)
        
        # 截断过长输入
        if len(sanitized) > cls.MAX_INPUT_LENGTH:
            sanitized = sanitized[:cls.MAX_INPUT_LENGTH]
        
        return sanitized.strip()
```

### 5.2 API 调用超时与重试

```python
# core/ai/error_handler.py
import asyncio
from dataclasses import dataclass

@dataclass
class NetworkConfig:
    timeout: int = 60          # 请求超时（秒）
    max_retries: int = 3       # 最大重试次数
    retry_delay: float = 2.0   # 重试延迟（秒）
    verify_ssl: bool = True    # SSL 验证

async def safe_api_call(func, config: NetworkConfig = None):
    """安全的 API 调用（超时 + 重试 + SSL）"""
    config = config or NetworkConfig()
    last_error = None
    
    for attempt in range(config.max_retries + 1):
        try:
            # 设置超时
            return await asyncio.wait_for(
                func(),
                timeout=config.timeout
            )
        except asyncio.TimeoutError:
            last_error = "请求超时"
            if attempt < config.max_retries:
                await asyncio.sleep(config.retry_delay * (2 ** attempt))
        except Exception as e:
            last_error = str(e)
            raise
    
    raise TimeoutError(f"API 调用失败（已重试 {config.max_retries} 次）: {last_error}")
```

## 6. 日志安全

### 6.1 敏感信息脱敏

```python
# utils/logger.py
import re
import logging

class SensitiveDataFilter(logging.Filter):
    """敏感数据过滤器 - 脱敏日志中的敏感信息"""
    
    SENSITIVE_PATTERNS = [
        (r'(api[_-]?key["\s:=]+)[^"\s\']+', r'\1***REDACTED***'),
        (r'(sk-[a-zA-Z0-9]{20,})', 'sk-***REDACTED***'),
        (r'(secret["\s:=]+)[^"\s\']+', r'\1***REDACTED***'),
        (r'(password["\s:=]+)[^"\s\']+', r'\1***REDACTED***'),
        (r'(token["\s:=]+)[^"\s\']+', r'\1***REDACTED***'),
        (r'(auth["\s:=]+)[^"\s\']+', r'\1***REDACTED***'),
        (r'Bearer\s+[a-zA-Z0-9._-]+', 'Bearer ***REDACTED***'),
    ]
    
    def filter(self, record):
        if hasattr(record, 'msg') and isinstance(record.msg, str):
            msg = record.msg
            for pattern, replacement in self.SENSITIVE_PATTERNS:
                msg = re.sub(pattern, replacement, msg, flags=re.IGNORECASE)
            record.msg = msg
        
        if hasattr(record, 'args') and record.args:
            # 处理 args 中的敏感信息
            filtered_args = []
            for arg in record.args:
                if isinstance(arg, str):
                    for pattern, replacement in self.SENSITIVE_PATTERNS:
                        arg = re.sub(pattern, replacement, arg, flags=re.IGNORECASE)
                filtered_args.append(arg)
            record.args = tuple(filtered_args)
        
        return True

# 在应用启动时配置
logger = logging.getLogger('novel_writer')
logger.addFilter(SensitiveDataFilter())
```

## 8. 插件安全

### 8.1 沙箱执行

```python
# core/plugins/sandbox.py
import builtins
from typing import Dict, Any

class PluginSandbox:
    """插件沙箱 - 限制插件可访问的 API"""
    
    # 白名单：插件允许访问的模块和函数
    ALLOWED_MODULES = {
        'json', 're', 'math', 'datetime',  # 标准库
        'typing', 'dataclasses',
    }
    
    ALLOWED_BUILTINS = {
        'print', 'len', 'range', 'int', 'str',
        'list', 'dict', 'tuple', 'bool', 'float',
        'min', 'max', 'sum', 'sorted', 'enumerate',
        'zip', 'map', 'filter', 'type', 'isinstance',
    }
    
    @classmethod
    def create_sandbox_globals(cls) -> Dict[str, Any]:
        """创建沙箱全局命名空间"""
        
        safe_builtins = {}
        for name in cls.ALLOWED_BUILTINS:
            if hasattr(builtins, name):
                safe_builtins[name] = getattr(builtins, name)
        
        return {
            '__builtins__': safe_builtins,
            '__import__': cls._safe_import,
        }
    
    @classmethod
    def _safe_import(cls, name, *args, **kwargs):
        """安全的 import（仅白名单模块）"""
        if name not in cls.ALLOWED_MODULES:
            raise ImportError(f"插件不允许导入模块: {name}")
        return __import__(name, *args, **kwargs)
```

from enum import Enum
from dataclasses import dataclass

class PluginPermission(Enum):
    """插件权限"""
    # 文件系统
    READ_PROJECT = "read_project"      # 读取项目数据
    WRITE_PROJECT = "write_project"   # 写入项目数据
    READ_FILE = "read_file"           # 读取任意文件
    WRITE_FILE = "write_file"         # 写入任意文件
    
    # 网络
    HTTP_REQUEST = "http_request"     # HTTP 请求
    AI_CALL = "ai_call"               # 调用 AI
    
    # 系统
    EXEC_COMMAND = "exec_command"     # 执行命令
    READ_ENV = "read_env"             # 读取环境变量
    
    # UI
    UI_MENU = "ui_menu"              # 扩展菜单
    UI_SIDEBAR = "ui_sidebar"        # 扩展侧边栏

@dataclass
class PluginPermissions:
    """插件权限集"""
    permissions: set
    
    def can(self, permission: PluginPermission) -> bool:
        return permission in self.permissions
    
    def require(self, permission: PluginPermission):
        if not self.can(permission):
            raise PermissionError(f"插件无权执行此操作: {permission.value}")

# 权限等级
BASIC_PERMISSIONS = {

## 12. 安全总结

### 12.1 安全设计亮点

| 特性 | 说明 |
|------|------|
| **API Key 加密存储** | 使用 PBKDF2 + AES-256 加密 |
| **统一 HTTPS** | 所有 AI API 调用强制 HTTPS |
| **输入验证** | 用户输入长度、模式、注入检查 |
| **日志脱敏** | API Key、密码等自动替换为 `***REDACTED***` |
| **插件沙箱** | 限制插件可访问的 API 和模块 |
| **权限管理** | 插件的细粒度权限控制 |
| **数据本地化** | 所有数据存储在本地，不上传云端 |
| **安全默认值** | 文件权限、超时、SSL 验证均有安全默认值 |

### 12.2 安全改进对比（与原版）

| 维度 | 原版 | PySide6 版 |
|------|------|-----------|
| API Key 存储 | 环境变量/配置文件 | 加密写入数据库 |
| 输入验证 | 无 | 长度、模式、注入检查 |
| 日志脱敏 | 无 | 自动脱敏 |
| 插件权限 | 完全信任 | 白名单 + 权限控制 |
| 文件权限 | 默认 | 严格限制 |
| 沙箱 | 无 | 插件沙箱 |

### 12.3 持续安全

1. **依赖更新**：定期运行 `pip list --outdated`，优先更新涉及网络的库
2. **安全审计**：每季度运行一次安全扫描
3. **漏洞监控**：关注 PySide6、SQLAlchemy 等核心依赖的安全公告
4. **备份验证**：定期测试备份恢复流程
