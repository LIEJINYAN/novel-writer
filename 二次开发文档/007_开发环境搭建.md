# 开发环境搭建

## 1. 系统要求

### 1.1 硬件要求

| 硬件 | 最低要求 | 推荐要求 |
|------|---------|---------|
| **内存** | 4GB RAM | 8GB+ RAM |
| **磁盘空间** | 1GB 可用空间 | 5GB+ 可用空间 |
| **处理器** | 双核 2.0GHz | 四核 2.5GHz+ |

### 1.2 软件要求

| 软件 | 最低版本 | 推荐版本 | 说明 |
|------|---------|---------|------|
| **Python** | 3.10 | 3.11+ | 核心运行环境 |
| **pip** | 21.0 | 24.x | Python 包管理器 |
| **Git** | 2.0 | 最新版 | 版本控制 |
| **Node.js** | 18.0 | 20.x LTS | 构建工具依赖 |

### 1.3 操作系统支持

| 操作系统 | 状态 | 说明 |
|---------|------|------|
| **Windows 10/11** | ✅ 正式支持 | PowerShell 5.1+ |
| **macOS 12+** | ✅ 正式支持 | Intel/Apple Silicon |
| **Linux (Ubuntu 20.04+)** | ✅ 正式支持 | 主要开发平台 |
| **Linux (其他发行版)** | ⚠️ 社区支持 | 需自行解决依赖 |

## 11. 构建与打包

### 11.1 PyInstaller 打包

```bash
# 安装 PyInstaller
pip install pyinstaller

# 一键打包（Windows 生成 exe）
pyinstaller --windowed --name "NovelWriter" --icon app/resources/icon.ico app/main.py

# 常用参数说明
pyinstaller \
  --windowed \           # 不显示控制台窗口
  --onefile \            # 打包成单个 exe 文件
  --name "NovelWriter" \ # 输出文件名
  --icon icon.ico \      # 应用图标
  --add-data "ui/styles;ui/styles" \  # 包含静态资源
  --hidden-import PySide6.QtXml \     # 确保 Qt 模块被打包
  app/main.py
```

### 11.2 使用 spec 文件（推荐）

```python
# novel-writer.spec
# -*- mode: python ; coding: utf-8 -*-

a = Analysis(
    ['app/main.py'],
    pathex=[],
    binaries=[],
    datas=[
        ('ui/styles', 'ui/styles'),
        ('ui/icons', 'ui/icons'),
    ],
    hiddenimports=[
        'PySide6.QtXml',
        'sqlalchemy.ext.asyncio',
    ],
    hookspath=[],
    hooksconfig={},
    runtime_hooks=[],
    excludes=[],
    noarchive=False,
)

pyz = PYZ(a.pure)

exe = EXE(
    pyz,
    a.scripts,
    a.binaries,
    a.datas,
    [],
    name='NovelWriter',
    debug=False,
    bootloader_ignore_signals=False,
    strip=False,
    upx=True,
    upx_exclude=[],
    runtime_tmpdir=None,
    console=False,          # Windows 下隐藏控制台
    disable_windowed_traceback=False,
    argv_emulation=False,
    target_arch=None,
    codesign_identity=None,
    entitlements_file=None,
    icon='app/resources/icon.ico',
)

app = BUNDLE(
    exe,
    name='NovelWriter.app',
    icon='app/resources/icon.icns',
    bundle_identifier='com.novelwriter.app',
)
```

### 11.3 构建命令

```bash
# 使用 spec 文件构建
pyinstaller novel-writer.spec

# 构建完成后，可执行文件位于 dist/ 目录
# Windows: dist/NovelWriter/NovelWriter.exe
# macOS: dist/NovelWriter.app
# Linux: dist/NovelWriter/
```

## 13. 常见安装问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| `pip install 失败` | 网络问题 | 使用国内镜像：`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt` |
| `PySide6 安装慢` | 文件较大 | 使用缓存或预编译 wheel |
| `venv 激活失败` | PowerShell 策略 | `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser` |
| `缺少 VC++ 运行时` | Windows 缺少 C++ 库 | 安装 Visual C++ Redistributable |
| `Ollama 连接失败` | Ollama 未启动 | 启动 Ollama 服务：`ollama serve` |
| `SQLite 版本过低` | 系统 SQLite 旧 | 升级 SQLite 或使用 Python 内置版本 |
| `导入原版项目失败` | 格式不兼容 | 手动转换或等待兼容更新 |
