## 4. 许可证分类与风险说明

### 4.1 许可证类型占比

```
┌─────────────────────────────────────────────────────────────┐
│ 运行时依赖许可证风险评估                                      │
├─────────────────────────────────────────────────────────────┤
│ MIT (宽松):             10/19 = 53%                         │
│ Apache-2.0 (宽松+专利):   4/19 = 21%                         │
│ BSD-3-Clause (宽松):      4/19 = 21%                         │
│ LGPL (弱copyleft):       1/19 = 5%                          │
│ 公共领域:                 0/19 = 0%                          │
├─────────────────────────────────────────────────────────────┤
│ 总体风险等级: 🟢 安全                                        │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 许可证类型说明

| 类型 | 许可证 | 说明 |
|------|--------|------|
| **宽松许可证** | MIT / BSD / Apache-2.0 | 允许任意使用、修改、分发，只需保留版权声明 |
| **弱 copyleft** | LGPL-2.1 / LGPL-3.0 | 修改库本身需开源，但动态链接的应用程序无需开源 |
| **强 copyleft** | GPL-2.0 / GPL-3.0 / AGPL-3.0 | 本项目不涉及 |
| **专有** | 各 API 服务 | 调用 API 按服务提供商条款执行 |

### 4.3 本项目中不存在的许可证风险

以下常见传染性许可证在本项目中**不存在**：
- GPL-2.0 / GPL-3.0
- AGPL-3.0
- MPL-2.0 (Mozilla Public License)
- EUPL (European Union Public License)

## 7. 依赖管理最佳实践

### 7.1 重新生成此清单

```bash
# 安装 pip-licenses
pip install pip-licenses

# 生成许可证报告
pip-licenses --format=markdown --with-urls --with-system

# 仅运行时依赖
pip-licenses --format=markdown --with-urls --no-dev

# 排除 pip-licenses 自身
pip-licenses --format=json --ignore-packages=pip-licenses > licenses.json
```

### 7.2 依赖安全扫描

```bash
# 安装 safety
pip install safety

# 检查已知漏洞
safety check -r requirements.txt

# 输出 JSON 报告
safety check -r requirements.txt --json > safety_report.json

# 更新 requirements.txt
pip list --outdated
pip freeze > requirements.txt
```

### 7.3 GitHub Actions 自动检查

```yaml
# .github/workflows/license-check.yml
name: License Check

on:
  push:
    branches: [main]
  pull_request:

jobs:
  license-check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.11"
      
      - run: pip install pip-licenses safety
      - run: pip-licenses --format=markdown --no-dev
      - run: safety check -r requirements.txt
```
