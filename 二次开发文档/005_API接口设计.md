# API 接口设计（内部）

## 1. 接口概述

PySide6 版的"接口"与原版有本质区别。原版是 CLI 工具，通过终端命令和 AI 助手斜杠命令交互；PySide6 版是桌面应用，所有功能通过**内部 Python API（服务层接口）**提供，UI 层调用这些接口完成业务操作。

### 1.1 接口分类

| 类别 | 说明 | 调用方 |
|------|------|--------|
| **项目服务接口** | 项目 CRUD、导入导出 | UI 层（菜单、对话框） |
| **编辑器服务接口** | 内容编辑、自动保存、格式化 | UI 层（编辑器组件） |
| **AI 服务接口** | AI 对话、续写、润色、分析 | UI 层（AI 面板、菜单） |
| **追踪服务接口** | 情节、角色、关系、时间线管理 | UI 层（追踪面板） |
| **插件服务接口** | 插件加载、管理、扩展点调用 | UI 层 + 核心层 |
| **AI 提供商接口** | 统一的 AI 调用接口 | AI 服务层 |
| **数据访问接口** | 数据库 CRUD 操作 | 服务层 |

### 1.2 设计原则

1. **接口与实现分离**：服务层定义接口，核心层提供实现
2. **异步优先**：耗时操作（AI 调用、大文件读写）使用 async/await
3. **信号通知**：状态变化通过 Qt 信号槽通知 UI
4. **事务安全**：数据库操作使用事务，确保一致性
5. **错误传播**：异常向上传播，UI 层统一处理

```python
# services/ai_service.py
from typing import AsyncIterator, Optional, List
from PySide6.QtCore import QObject, Signal

class AIService(QObject):
    """AI 服务 - 连接 UI 和 AI 引擎"""
    
    # 信号
    ai_started = Signal(str)              # AI 调用开始 (operation)
    ai_finished = Signal(str)            # AI 调用完成
    ai_error = Signal(str, str)          # AI 错误 (error_type, message)
    ai_stream_chunk = Signal(str)        # 流式输出块 (text)
    ai_progress = Signal(int, int)       # 进度 (current, total)
    
    def __init__(self, ai_manager, writing_service, analysis_service):
        super().__init__()
        self.ai_manager = ai_manager
        self.writing_service = writing_service
        self.analysis_service = analysis_service
    
    # ========== 写作 AI ==========
    
    async def continue_writing(
        self,
        project_id: int,
        chapter_id: int,
        target_words: int = 4000
    ) -> AsyncIterator[str]:
        """AI 续写（流式）"""
        self.ai_started.emit("continue_writing")
        try:
            async for chunk in self.writing_service.continue_writing(
                project_id, chapter_id, target_words
            ):
                self.ai_stream_chunk.emit(chunk)
                yield chunk
            self.ai_finished.emit("continue_writing")
        except Exception as e:
            self.ai_error.emit("writing", str(e))
    
    async def polish_text(self, text: str, style: str = "更生动") -> str:
        """AI 润色"""
        self.ai_started.emit("polish")
        try:
            result = await self.writing_service.polish_text(text, style)
            self.ai_finished.emit("polish")
            return result
        except Exception as e:
            self.ai_error.emit("polish", str(e))
            return ""
    
    async def rewrite_text(self, text: str, instruction: str) -> str:
        """AI 改写"""
        ...
    
    async def generate_dialogue(
        self,
        characters: List[str],
        scene: str,
        goal: str
    ) -> str:
        """生成对话"""
        ...
    
    async def generate_outline(self, idea: str, method: str) -> str:
        """生成大纲"""
        ...
    
    # ========== 创作流程 AI ==========
    
    async def create_constitution(self, params: dict) -> str:

    async def create_specification(self, user_input: str, level: int) -> str:
        """创建故事规格"""

    async def create_plan(self, specification: str) -> str:

    async def generate_tasks(self, plan: str) -> str:

        """专家模式分析"""
        ...
    
    async def check_consistency(self, project_id: int) -> dict:
        """一致性检查"""
        ...
    
    # ========== AI 对话 ==========
    
    async def chat(
        self,
        message: str,
        conversation_id: int = None,
        context: dict = None
    ) -> AsyncIterator[str]:
        """AI 对话（流式）"""
        ...

        """获取对话历史"""
        ...
```

## 7. AI 提供商接口

### 7.1 统一接口

```python
# core/ai/providers/base.py
from abc import ABC, abstractmethod
from typing import AsyncIterator, List
from dataclasses import dataclass

@dataclass
class Message:
    role: str       # system / user / assistant
    content: str

@dataclass
class AIConfig:
    temperature: float = 0.7
    max_tokens: int = 4000
    top_p: float = 1.0
    frequency_penalty: float = 0.0
    presence_penalty: float = 0.0
    stop: List[str] = None

@dataclass
class AIResponse:
    content: str
    model: str
    usage: dict       # {prompt_tokens, completion_tokens, total_tokens}
    finish_reason: str

class AIProvider(ABC):
    """AI 提供商抽象接口"""
    
    name: str
    display_name: str
    
    @abstractmethod
    async def chat(
        self,
        messages: List[Message],
        config: AIConfig = None,
        model: str = None
    ) -> AIResponse:
        """同步对话"""
        ...
    
    @abstractmethod
    async def chat_stream(
        self,
        messages: List[Message],
        config: AIConfig = None,
        model: str = None
    ) -> AsyncIterator[str]:
        """流式对话"""
        ...
    
    @abstractmethod
    def get_available_models(self) -> List[str]:
        """获取可用模型"""
        ...
    
    @abstractmethod
    def validate_config(self) -> bool:
        """验证配置"""
        ...
```

### 9.2 信号使用示例

```python
# 在服务层发送信号
class EditorService:
    def save_chapter(self, chapter_id, content):
        # 保存逻辑...
        signal_bus.content_saved.emit(chapter_id)
        signal_bus.status_message.emit(f"章节 {chapter_id} 已保存")

# 在 UI 层接收信号
class MainWindow(QMainWindow):
    def __init__(self):
        super().__init__()
        signal_bus.content_saved.connect(self.on_content_saved)
        signal_bus.status_message.connect(self.statusBar().showMessage)
    
    def on_content_saved(self, chapter_id):
        # 更新 UI 显示
```
