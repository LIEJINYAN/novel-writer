# 插件开发指南

## 1. 插件系统概述

PySide6 版的插件系统与原版有本质区别。原版通过向 AI 助手目录注入命令文件实现"插件"，而 PySide6 版拥有真正的 **Python 插件引擎**，插件可以：

- 注册新的 **AI 提供商**
- 扩展 **菜单和侧边栏面板**
- 添加 **导出格式**
- 注册 **写作方法论**
- 注册 **追踪器**
- 通过 **沙箱安全执行**
- 使用 **细粒度权限控制**

### 1.1 与原版对比

| 维度 | 原版（命令注入） | PySide6 版（Python 插件） |
|------|----------------|-------------------------|
| **插件语言** | Markdown/TOML | Python |
| **扩展能力** | 仅斜杠命令 | 菜单/面板/提供商/导出等 |
| **权限控制** | 无 | 4 级权限管理 |
| **执行环境** | AI 助手内部 | 插件沙箱 |
| **API 能力** | 有限 | 完整的 PySide6 API |
| **UI 扩展** | 不支持 | 支持（菜单、侧边栏） |
| **依赖管理** | 无 | pip 依赖 |

## 2. 插件结构

### 2.1 目录布局

```
plugins/
└── your-plugin-name/           # 插件目录（使用蛇形命名）
    ├── __init__.py             # 必需：插件入口
    ├── plugin.yaml             # 必需：插件配置
    ├── README.md               # 推荐：插件说明
    ├── commands/               # 可选：命令模块
    │   ├── __init__.py
    │   └── your_command.py
    ├── panels/                 # 可选：UI 面板
    │   ├── __init__.py
    │   └── your_panel.py
    ├── providers/              # 可选：AI 提供商
    │   ├── __init__.py
    │   └── your_provider.py
    ├── exporters/              # 可选：导出格式
    │   ├── __init__.py
    │   └── your_exporter.py
    ├── resources/              # 可选：资源文件
    │   ├── icons/
    │   └── templates/
    └── requirements.txt       # 可选：插件依赖
```

## 3. 插件基类

```python
# core/plugins/base.py
from abc import ABC, abstractmethod
from typing import Dict, List, Optional, Callable, Any
from PySide6.QtWidgets import QWidget


class PluginBase(ABC):
    """插件抽象基类。"""
    
    # 基本信息（子类必须重写）
    name: str = ""
    display_name: str = ""
    version: str = ""
    description: str = ""
    author: str = ""
    license: str = "MIT"
    
    # 运行时状态
    is_loaded: bool = False
    is_enabled: bool = False
    
    def __init__(self, plugin_manager) -> None:
        self.manager = plugin_manager
        self._commands: Dict[str, Callable] = {}
        self._panels: Dict[str, type] = {}
        self._exporters: Dict[str, type] = {}
    
    @abstractmethod
    def on_load(self) -> None:
        """插件加载（必须实现）。"""
        ...
    
    def on_unload(self) -> None:
        """插件卸载（可选重写）。"""
        pass
    
    def on_enable(self) -> None:
        """插件启用（可选重写）。"""
        pass
    
    def on_disable(self) -> None:
        """插件禁用（可选重写）。"""
        pass
    
    # ========== 注册方法 ==========
    
    def register_command(
        self,
        command_id: str,
        handler: Callable,
    ) -> None:
        """注册命令。"""
        self._commands[command_id] = handler
    
    def register_panel(self, panel_id: str, panel_class: type) -> None:
        """注册侧边栏面板。"""
        self._panels[panel_id] = panel_class
    
    def register_exporter(self, format_name: str, exporter_class: type) -> None:
        """注册导出格式。"""
        self._exporters[format_name] = exporter_class
    
    # ========== 辅助方法 ==========
    
    def get_data_dir(self) -> str:
        """获取插件数据目录。"""
        return self.manager.get_plugin_data_dir(self.name)
    
    def log_info(self, message: str) -> None:
        """记录信息日志。"""
        self.manager.log_info(f"[{self.name}] {message}")
    
    def log_error(self, message: str) -> None:
        """记录错误日志。"""
        self.manager.log_error(f"[{self.name}] {message}")
```

### 4.3 注册导出格式

```python
# exporters/my_exporter.py
from pathlib import Path
from typing import List
from models import Chapter


class MyExporter:
    """自定义导出格式。"""
    
    name = "my-format"
    display_name = "我的格式"
    extension = ".myfmt"
    
    def export(
        self,
        output_path: str,
        chapters: List[Chapter],
        options: dict = None,
    ) -> str:
        """导出文件。
        
        Args:
            output_path: 输出文件路径。
            chapters: 要导出的章节列表。
            options: 导出选项。
        
        Returns:
            输出文件的路径。
        """
        with open(output_path, "w", encoding="utf-8") as f:
            for chapter in chapters:
                f.write(f"# {chapter.title}\n\n")
                f.write(chapter.content)
                f.write("\n\n")
        
        return output_path


# 在插件中注册
def on_load(self) -> None:
    self.register_exporter("my-format", MyExporter)
```

### 5.3 权限等级

| 权限等级 | 包含的权限 | 说明 |
|---------|-----------|------|
| **BASIC** | read_project, ui_menu | 只读 + 菜单扩展 |
| **STANDARD** | + read_file, ui_sidebar | 文件读取 + 面板 |
| **ADVANCED** | + write_project, http_request | 写入 + 网络 |
| **FULL** | + write_file, ai_call, read_env | 完全访问 |

## 8. 开发规范

### 8.1 命名规范

- **插件名称**：蛇形命名（`my-plugin-name`）
- **Python 模块**：蛇形命名（`my_plugin_name`）
- **类名**：PascalCase（`MyPluginName`）
- **命令 ID**：短横线命名（`my-command`）

### 8.2 文件规范

```
plugins/my-plugin-name/
├── __init__.py     # 插件入口（必需）
├── plugin.yaml     # 插件配置（必需）
└── ...
```

`__init__.py` 中必须包含一个继承 `PluginBase` 的类，且类名与文件名对应。

### 8.3 版本规范

使用语义化版本（SemVer）：
- **MAJOR**: 不兼容的 API 变更
- **MINOR**: 向后兼容的功能新增
- **PATCH**: 向后兼容的 bug 修复

### 8.4 安全性

1. **不要在代码中硬编码 API Key**
2. **使用插件沙箱**：无法访问文件系统外的数据
3. **权限最小化**：只申请必要的权限
4. **输入验证**：对所有用户输入进行验证
5. **数据清理**：用完敏感数据后清理内存

## 10. 完整示例

### 10.1 翻译插件示例

```python
# plugins/translate/__init__.py
"""中英文翻译插件。"""

from core.plugins.base import PluginBase


class TranslatePlugin(PluginBase):
    """翻译插件 - 提供中英文互译功能。"""
    
    name = "translate"
    display_name = "翻译插件"
    version = "1.0.0"
    description = "中英文翻译和润色"
    author = "Novel Writer Team"
    
    def on_load(self) -> None:
        # 注册命令
        self.register_command("translate", self._translate)
        self.register_command("translate-chapter", self._translate_chapter)
        self.register_command("polish", self._polish)
        
        self.log_info("翻译插件已加载")
    
    async def _translate(self, args: str, context: dict) -> str:
        """翻译文本。"""
        ai_service = context.get("ai_service")
        if not ai_service:
            return "错误: AI 服务不可用"
        
        result = await ai_service.generate(
            system_prompt="你是一个专业的翻译助手。",
            prompt=f"将以下中文翻译为英文：\n\n{args}",
        )
        return result.content
    
    async def _translate_chapter(self, args: str, context: dict) -> str:
        """翻译整个章节。"""
        return "章节翻译功能..."
    
    async def _polish(self, args: str, context: dict) -> str:
        """英文润色。"""
        return "英文润色功能..."
```

## 12. API 参考

### 12.1 PluginBase 方法

| 方法 | 说明 |
|------|------|
| `on_load()` | 插件加载时调用（必须实现） |
| `on_unload()` | 插件卸载时调用 |
| `on_enable()` | 插件启用时调用 |
| `on_disable()` | 插件禁用时调用 |
| `register_command(id, handler)` | 注册命令 |
| `register_panel(id, panel_class)` | 注册侧边栏面板 |
| `register_exporter(name, exporter_class)` | 注册导出格式 |
| `get_data_dir()` | 获取插件数据目录 |
| `log_info(msg)` | 记录信息日志 |
| `log_error(msg)` | 记录错误日志 |

### 12.2 可用信号

| 信号 | 参数 | 说明 |
|------|------|------|
| `signal_bus.project_opened` | int (project_id) | 项目打开 |
| `signal_bus.project_closed` | 无 | 项目关闭 |
| `signal_bus.content_saved` | int (chapter_id) | 章节保存 |
| `signal_bus.ai_finished` | str (operation) | AI 调用完成 |
| `signal_bus.status_message` | str (message) | 状态栏消息 |
| `signal_bus.plugin_changed` | str (plugin_name) | 插件状态变化 |
