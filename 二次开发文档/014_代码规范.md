# 代码规范

> **技术栈**: Python 3.10+ / PySide6  
> **最后更新**: 2026-07-02

## 1. 代码风格概述

本项目使用 **Black** 作为代码格式化工具，**Ruff** 作为代码检查工具（替代 ESLint + isort + flake8），**Mypy** 作为类型检查器。

### 1.1 核心规范速览

| 规范项 | 规则 |
|--------|------|
| **缩进** | 4 个空格（Python 标准） |
| **行宽** | 100 字符 |
| **引号** | 双引号（Black 默认） |
| **行尾分号** | 不使用 |
| **类型注解** | 必须（公共 API） |
| **import 排序** | Ruff 自动处理 |
| **末尾换行** | 文件末尾空一行 |

### 3.1 Python

| 类别 | 风格 | 示例 |
|------|------|------|
| 变量名 | `snake_case` | `project_name`, `chapter_count` |
| 函数名 | `snake_case` | `load_chapter()`, `count_words()` |
| 类名 | `PascalCase` | `MainWindow`, `TextEditor` |

| 类别 | 风格 | 示例 |

## 4. 代码示例

### 4.1 类定义

```python
# ✅ 正确
from typing import Optional
from PySide6.QtCore import QObject, Signal


class ChapterService(QObject):
    """章节服务 - 管理章节的 CRUD 操作。"""

    chapter_created = Signal(int)   # (chapter_id)
    chapter_deleted = Signal(int)   # (chapter_id)

    def __init__(self, db_session) -> None:
        super().__init__()
        self._session = db_session

    def create_chapter(
        self,
        project_id: int,
        title: str,
        content: str = "",
    ) -> "Chapter":
        """创建新章节。

        Args:
            project_id: 所属项目 ID。
            title: 章节标题。
            content: 章节正文（可选）。

        Returns:
            创建的 Chapter 实例。
        """
        chapter = Chapter(
            project_id=project_id,
            chapter_number=self._get_next_number(project_id),
            title=title,
            content=content,
        )
        self._session.add(chapter)
        self._session.commit()
        self.chapter_created.emit(chapter.id)
        return chapter

    def _get_next_number(self, project_id: int) -> int:
        """获取项目下一个可用章节号。"""
        last = (
            self._session.query(Chapter.chapter_number)
            .filter_by(project_id=project_id)
            .order_by(Chapter.chapter_number.desc())
            .first()
        )
        return (last[0] + 1) if last else 1
```

### 4.4 异常处理

```python
# ✅ 正确
async def safe_ai_call(self, prompt: str) -> str:
    """安全的 AI 调用（含错误处理）。"""
    try:
        return await self._ai.generate(prompt)
    except AuthError:
        signal_bus.ai_error.emit("认证失败，请检查 API Key 设置")
        return ""
    except RateLimitError:
        signal_bus.ai_error.emit("请求过于频繁，请稍后重试")
        return ""
    except Exception as e:
        logger.exception("AI 调用异常")
        signal_bus.ai_error.emit(f"AI 调用失败: {e}")
        return ""


# ❌ 错误
def ai_call(prompt):  # 无类型注解

        return self.ai.generate(prompt)
    except:  # 裸 except，隐藏所有错误
```

## 10. 迁移说明（从原版 TypeScript）

若将原版 TypeScript 代码迁移到 Python，注意以下差异：

| TypeScript | Python | 说明 |
|-----------|--------|------|
| `camelCase` | `snake_case` | 变量和函数名 |
| `const` / `let` | 直接赋值 | 无常量声明关键字 |
| `interface` | `dataclass` / `TypedDict` | 数据结构定义 |
| `async/await` | `async/await` | 异步编程类似 |
| `throw` | `raise` | 异常抛出 |
| `null` / `undefined` | `None` | 空值 |
| `console.log()` | `logger.info()` | 日志输出 |
| 行尾分号 | 无分号 | Python 无分号 |
| 2 空格缩进 | 4 空格缩进 | Python 标准 |
